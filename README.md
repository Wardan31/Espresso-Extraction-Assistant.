# Espresso-Extraction-Assistant.
"A Java console app that diagnoses espresso extraction issues based on your machine, grinder, and shot metrics."
## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Make sure grinders.txt and EspressoMachines.txt 
   are in the root project folder
4. Run Main.java

## Features
- Search for your espresso machine and grinder
- Input shot metrics (dose, yield, time)
- Get diagnosis and tips based on extraction data

## Built With
- Java
- OOP principles (Inheritance, Polymorphism, Encapsulation)
